# REFERENCE

This directory is designated for project reference materials, documentation, and related academic resources.

## Purpose

Store project-related documents such as:
- Project proposals and presentations
- Research papers and literature
- Technical documentation
- Meeting notes and appendices
- Sample proposals and examples

## Note

Reference documents from the original project structure were removed during reorganization. If you need to restore them, they can be recovered from the git history:

```bash
git checkout HEAD -- reference/
```

## Recommended Contents

- Project proposals (PDF/DOCX)
- Academic papers and references
- Design documents
- Architecture diagrams
- Meeting notes and minutes
- External documentation
